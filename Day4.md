# Day 4 - Cybersecurity Journey

Today I learned about networking concepts.

Topics covered:
- IP Address and MAC Address
- How devices communicate in a network
- Ping (used to check connectivity)
- MAC Address Spoofing

What I understood:
IP address is used to identify devices on a network, while MAC address is a unique physical identifier of a device.

Ping is used to check whether a system is reachable.

MAC address spoofing is a technique where an attacker changes their MAC address to impersonate another device or hide their identity.
