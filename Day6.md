# Day 6 - Cybersecurity Journey

Today I learned advanced networking concepts.

Topics covered:
- Static IP and Dynamic IP
- IP address classes and subnetting
- NAT (Network Address Translation)
- Protocols and ports
- OSI model and TCP/IP model
- TCP vs UDP

What I understood:
IP addressing helps identify devices in a network, and subnetting helps in managing networks efficiently.

NAT allows private networks to communicate with the internet using a public IP.

Protocols define communication rules, and ports act as entry points for services.

TCP ensures reliable communication, while UDP provides faster but less reliable transmission.
