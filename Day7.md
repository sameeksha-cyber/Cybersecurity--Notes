# Day 7 - Cybersecurity Journey

Today I started learning web security concepts.

Topics covered:
- Introduction to Cross-Site Scripting (XSS)

What I understood:
XSS is a vulnerability where an attacker injects malicious scripts into a website, which then run in the victim’s browser.

This can allow attackers to steal data, hijack sessions, or perform actions on behalf of the user.
